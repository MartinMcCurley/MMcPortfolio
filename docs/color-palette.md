# Color Palette - Martin McCurley Portfolio

## Primary Dark Theme
Based on research from Perplexity.ai and Linear.app aesthetics.

### Base Colors
```css
--background-primary: #1A1A1A;    /* Soft dark gray for main background */
--background-secondary: #202020;   /* Slightly lighter for cards/sections */
--background-tertiary: #252525;    /* Even lighter for hover states */
```

### Text Colors
```css
--text-primary: #EEEEEE;          /* High contrast light gray for main text */
--text-secondary: #B3B3B3;        /* Medium gray for secondary text */
--text-muted: #808080;            /* Lighter gray for captions/metadata */
```

### Accent Colors (Futuristic)
```css
--accent-primary: #3291FF;        /* Electric blue - primary CTA/highlights */
--accent-secondary: #00D9FF;      /* Cool cyan - secondary highlights */
--accent-hover: #5AA3FF;          /* Lighter blue for hover states */
```

### Utility Colors
```css
--border-subtle: #333333;         /* Subtle borders */
--shadow-light: rgba(50, 145, 255, 0.1);  /* Subtle blue glow */
--shadow-dark: rgba(0, 0, 0, 0.3); /* Dark shadows */
```

## Design Principles
- **High Contrast**: Ensure WCAG AA compliance (4.5:1 ratio minimum)
- **Consistent Application**: Use colors systematically across all elements
- **Subtle Futurism**: Accent colors provide the "futuristic" feel without being overwhelming
- **Performance**: Minimal color palette reduces CSS complexity
